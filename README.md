# GitHub Pages source

`githubpage/` は GitHub Pages 用の静的サイト一式です。

含まれる公開ページ:

- `/`
- `/support/`
- `/nekocafe/privacy-policy/`
- `/uric-acid-note/privacy-policy/`
- `/online-lottery/privacy-policy/`
- `/simple-shared-budget/privacy-policy/`

運用メモ:

- `nk-32.github.io` リポジトリで使う場合は、このディレクトリの中身をサイトの公開ルートに置く想定です。
- GitHub Pages を branch deploy で使う場合、標準設定で選べる公開元は `/` または `/docs` です。
